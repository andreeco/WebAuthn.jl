### Registration Workflow

```mermaid
sequenceDiagram
    participant User
    participant Browser
    participant Authenticator
    participant Server

    Server->>Browser: Build registration options
    Browser->>Authenticator: navigator.credentials.create(options)
    Authenticator-->>Browser: attestationObject, clientDataJSON
    Browser->>Server: Submit attestationObject, clientDataJSON
    Server->>Server: Verify attestation and register credential
```

### Authentication Workflow

```mermaid
sequenceDiagram
    participant User
    participant Browser
    participant Authenticator
    participant Server

    Server->>Browser: Build authentication options
    Browser->>Authenticator: navigator.credentials.get(options)
    Authenticator-->>Browser: assertion data
    Browser->>Server: Submit assertion data
    Server->>Server: Verify signature, authenticator data
```
