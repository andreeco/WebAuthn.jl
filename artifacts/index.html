WebAuthn Passkey Demo

This demo lets you register and use passkeys. Choose a username or leave blank to be auto-assigned.

Register a new Passkey



Login using Passkey


Instructions
  • You may choose a username (email/handle), or leave it blank to get an auto-assigned one.
  • Remember your username for login if you provided one.
  • If you leave username blank at login, the server will attempt a usernameless login (if your authenticator supports it).
  • The "Device label" lets you assign a name to this passkey.
